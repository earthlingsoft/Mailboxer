Mailboxer

Mailboxer erstellt Intelligente Postfächer in Mail für Kontakte aus dem Adressbuch.

Obwohl das Mail Programm in Mac OS X bereits alle dafür notwendigen Bausteine enthält, ist es recht umständlich Intelligente Postfächer für bestimmte Personen oder Firmen anzulegen. Hierzu muß für jede dieser Personen das Intelligente Postfach von Hand angelegt und mit allen E-Mail Adressen dieser Person ausgestattet werden. Kommt eine neue Adresse hinzu, muß auch das Intelligente Postfach aktualisiert werden. Mailboxer hilft hierbei.

In Mailboxers Hauptfenster kann eine Gruppe aus dem Adressbuch (oder das gesamte Adressbuch) ausgewählt werden. Für jede darin enthaltene Visitenkarte legt Mailboxer dann ein Intelligentes Postfach an, das die Nachrichten von und an alle E-Mail Adressen dieser Person oder Firma enthält. Diese Intelligenten Postfächer werden dann in einem Order für Intelligente Postfächer in Mail eingefügt. Der Name dieses Ordners kann frei gewählt werden, wobei zu beachten ist, daß eventuell existierende Ordner mit demselben Namen gelöscht werden.

Technisch gesehen verändert Mailboxer die Datei ~/Library/Mail/SmartMailboxes.plist, von der zuvor eine Sicherungskopie mit dem Namen ,SmartMailboxes.plist (Vor Mailboxer)‘ angelegt wird. In diese Datei werden die neu erzeugten Intelligenten Postfächer eingefügt. Und Mail wird falls nötig neu gestartet, damit die neu eingefügten Intelligenten Postfächer auch angezeigt werden. Sollte es nötig sein, die Darstellungsoptionen für die Intelligenten Postfächer zu verändern, helfen die Einstellungen weiter.

Vielen Dank an Henrik Cederblad für Mailboxers Icon und an Ronald Leroux für die Französische Lokalisierung.

Enjoy.

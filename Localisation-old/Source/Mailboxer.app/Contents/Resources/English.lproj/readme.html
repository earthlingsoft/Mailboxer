Mailboxer

This little tool sets up a number of smart Mailboxes in Mail based on the contacts in your Address Book.

It would seem natural for Mail to come with this feature built in. After all, the hard work such as providing a system-wide Address Book and a Smart Mailbox feature in Mail has already been done by Apple. But this feature does not exist and thus the hassle of having create Smart Mailboxes for all the present and past e-mail addresses of each person in the Address Book is left to the user. And Mailboxer steps in right there.

Mailboxer’s main window lets you select whether you want a smart folder for each entry in your Address Book or whether you prefer limiting that to just the contacts of a single group of your Address Book. You can also choose a name for the Smart Folder Mailboxer inserts into your mailbox list. Please keep in mind that any existing Smart Folder of the same name will be deleted in the process.

Technically, all Mailboxer does is that it edits Mail’s setup for Smart Mailboxes to include this extra folder with the Smart Mailboxes it creates. In order to do that Mail needs to be quit and re-launched. The file it edits is ~/Library/Mail/SmartMailboxes.plist and a backup copy named ‘SmartMailboxes (Pre Mailboxer).plist’ will be created in the same location before editing the file. If you require additional customisation of the contents or sort-order of the Smart Mailboxes, the Preferences window is your friend.

Thanks to Henrik Cederblad for Mailboxer's icon and to Ronald Leroux for the French localisation.

Enjoy.
